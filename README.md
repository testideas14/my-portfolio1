# Portfolio Template Starter

This folder contains a simplified, ready-to-extend portfolio template.

## What is included

- Next.js + TypeScript setup
- Tailwind CSS v4 setup
- Single-page portfolio layout
- Base sections:
  - Hero
  - About
  - Projects
  - Skills
  - Other

## Quick start

1. Copy this `portfolio-template/` folder to a new location.
2. Install dependencies:
   - `npm install`
3. Start development:
   - `npm run dev`
4. Check [localhost:3000](http://localhost:3000)
